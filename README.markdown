# ~/.dotfiles

## Installation

``` sh
FRESH_LOCAL_SOURCE=JaredSharplin/.dotfiles bash <(curl -sL get.freshshell.com)
```

My dotfiles are managed by [fresh].

[fresh]: http://freshshell.com
